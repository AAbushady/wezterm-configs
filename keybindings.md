# WezTerm Keybindings

> On macOS, `Alt` = `Opt` and `Ctrl` = `Cmd` for clipboard/tab/font shortcuts.

## Tabs

| Windows/Linux | macOS | Action |
|---|---|---|
| `Ctrl+Shift+T` | `Cmd+T` | New tab |
| `Ctrl+Shift+W` | `Cmd+W` | Close tab |

## Panes

| Windows/Linux | macOS | Action |
|---|---|---|
| `Alt+\|` | `Opt+\|` | Split side by side |
| `Alt+_` | `Opt+_` | Split top/bottom |
| `Alt+Arrow` | `Opt+Arrow` | Move between panes |
| `Alt+Shift+Arrow` | `Opt+Shift+Arrow` | Resize active pane |
| `Alt+Shift+S` | `Opt+Shift+S` | Swap pane positions |
| `Alt+Shift+T` | `Opt+Shift+T` | Pane → its own tab |
| `Alt+Shift+M` | `Opt+Shift+M` | Tab → merge as pane (side by side) |
| `Alt+Shift+B` | `Opt+Shift+B` | Tab → merge as pane (top/bottom) |
| `Alt+Shift+Z` | `Opt+Shift+Z` | Zoom pane (toggle) |
| `Alt+Shift+W` | `Opt+Shift+W` | Close current pane |

## Workspaces

| Windows/Linux | macOS | Action |
|---|---|---|
| `Alt+Shift+L` | `Opt+Shift+L` | Pick a workspace layout (auto-generated from `hosts.lua`) |

## SSH

| Windows/Linux | macOS | Action |
|---|---|---|
| `Alt+S` | `Opt+S` | SSH launcher (open remote tab) |

## Clipboard

| Windows/Linux | macOS | Action |
|---|---|---|
| `Ctrl+C` | `Cmd+C` | Copy if selected, otherwise interrupt |
| `Ctrl+V` | `Cmd+V` | Paste from clipboard |

## Font Size

| Windows/Linux | macOS | Action |
|---|---|---|
| `Ctrl+=` | `Cmd+=` | Zoom in |
| `Ctrl+-` | `Cmd+-` | Zoom out |
| `Ctrl+0` | `Cmd+0` | Reset |
